## 搭建博客

今天用 Hugo + PaperMod 主题搭建了这个博客，托管在 GitHub Pages 上。

整个过程很简单：

1. Fork 模板仓库
2. 修改 `config.yml` 中的 `baseURL`
3. 配置 GitHub Actions 自动部署
4. 开始写文章
